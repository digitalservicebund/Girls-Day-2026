HTML

Was ist HTML?

HTML steht für HyperText Markup Language. Es ist die Sprache, mit der du den Inhalt einer Webseite beschreibst – also Text, Bilder, Links und Formulare.

HTML besteht aus sogenannten Tags. Ein Tag sagt dem Browser, was ein Element ist. Die meisten Tags haben ein öffnendes und ein schließendes Tag: 

<p>Das ist ein Absatz.</p>
<h1>Das ist eine Überschrift.</h1>

So sieht es im Browser aus:

Das ist eine Überschrift.

Das ist ein Absatz.

HTML

Die wichtigsten Tags

Tag Bedeutung
<h1> … <h6> Überschriften (h1 = groß, h6 = klein)
<p> Absatz (Paragraph)
<strong> Fett gedruckter Text
<em> Kursiver Text
<a href="…"> Ein Link
<img src="…"> Ein Bild
<ul> & <li> Liste mit Punkten
<div> Ein Container (zum Gruppieren)

Bilder im HTML

Bilder - Wie du sie hinzufügst

Bilder können über ein src-Attribut hinzugefügt werden. Dort kannst du zum Beispiel angeben wo ein Bild liegt. 


<img src="bilder/bild1.jpg"
     alt="Mein Foto" />

Das alt-Attribut beim Bild ist ein kurzer Beschreibungstext – wichtig für Menschen, die Screenreader benutzen.

CSS

Was ist CSS?

CSS steht für Cascading Style Sheets. Während HTML den Inhalt beschreibt, kümmert sich CSS um das Aussehen – Farben, Schriftarten, Abstände, Animationen.

Eine CSS-Regel besteht aus einem Selektor und einer oder mehreren Eigenschaften: 

p {
  color: hotpink;
  font-size: 20px;
  font-weight: bold;
}

So sieht dieser Text dann aus!

CSS

CSS-Klassen

Mit dem Attribut class kannst du einem HTML-Element eine CSS-Klasse geben. In eurer styles.css sind schon viele Klassen fertig – einfach draufschreiben und ausprobieren: 

<h2 class="pink">Rosa Überschrift</h2>
<p  class="big">Großer Text</p>
<p  class="small">Kleiner Text</p>

CSS kennt viele Eigenschaften. Hier die häufigsten:

Eigenschaft Bedeutung

Tipp

Hilfreiche Browser-Tricks

Browser-Entwicklertools öffnen
Drücke F12 oder Rechtsklick → Untersuchen. Du kannst dort live CSS ändern und sehen wie deine Seite aussieht – ohne etwas kaputtzumachen!
Seite neu laden
Nach jeder Änderung im Editor einfach Strg+S speichern und im Browser F5 drücken – schon siehst du deine Änderungen.
Fehler finden
Wenn etwas nicht klappt: prüfe, ob jedes öffnende Tag auch ein schließendes hat. Der Browser ist sehr verständnisvoll – aber vergessene </p> können das Layout durcheinanderbringen.

Bereit? Öffne deine Seite und leg los!
